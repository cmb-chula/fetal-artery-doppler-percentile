# fetal-doppler-percentile
Percentile predictor for fetal doppler parameters, with gestation age (GA) and fetal heart rate (FHR) as inputs

Currently supported parameters:
1. AT
2. ET
3. AT/ET
4. EDV
5. PDV
6. PEDRF
7. PEDRF/PSV
8. SNV
9. SNV/PSV
10. TSN
11 TSN/ET
